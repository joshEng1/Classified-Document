Portfolio Demo — Live Cloud Pipeline

Classify, cite, and redact documents with AI.

Upload a multi-page PDF or DOCX. The pipeline extracts content with Azure Document Intelligence, flags PII and unsafe risk with Gemini 3.0 Flash, and produces an audit-ready report with page-level citations and an optional redacted PDF.

API health
Portfolio demo — do not upload documents containing real personal or sensitive data. Uploaded files are processed by Azure Document Intelligence and Gemini 3.0 Flash in the cloud. This deployment exists solely for recruiter evaluation. The full offline-first version using local IBM Granite & llama.cpp models is available in the source repo.

Pipeline Architecture

A seamless flow from unstructured document to redacted intelligence.

1

Ingestion & OCR

Extract text & layouts natively or fallback to Azure Doc Intel for complex images.

2

Chunking

Slice text into overlapping context windows to maintain semantic boundaries.

3

LLM Analysis

gemini-flash 3.0 parses chunks to flag PII, assess risk, and assign classifications.

4

Report & Redact

Compile evidence, generate JSON audit, and overlay redaction blocks on PDF.

What I Built

  • Orchestration Node Server: SSE pipeline managing chunking, concurrency, and async LLM fallbacks.
  • Native PDF Redaction: In-browser Canvas + Node-canvas overlay generation by coordinate mapping.
  • Heuristic Filters: Primary regex & pattern matching for PII prior to AI verification.
